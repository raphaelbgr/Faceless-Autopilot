# US-016: Content Performance Comparison

## User Story
**As a user, I want to compare performance across different content types so that I can identify what works**

## Acceptance Criteria
- [ ] System shall compare performance across content types
- [ ] User can see performance metrics by format
- [ ] System shall identify top-performing content
- [ ] User can analyze performance trends
- [ ] System shall provide performance recommendations

## Definition of Done
- Performance comparison is accurate
- User can identify best-performing content
- Trends are clearly displayed
- Recommendations are helpful
- Comparison helps improve content strategy

## Story Points: 5

## Epic: Analytics & Optimization

## Dependencies
- Analytics system
- Content classification system
- Comparison system

## Risks
- Comparison may not be accurate
- User may not understand the data
- Recommendations may not be helpful
